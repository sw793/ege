# Задание 15. Дискретные множества
`
z.count(x)
`
— находим кол-во элементов x в множестве z.

`
z.insert(x)
`
— заносим элемент x в множество z (обычно используется в цикле).
- Если просят найти что-то **наибольшее**, то:
    - «x ∈ A» заменяется **1**
    - if ( (условие) **== 1**) { ... }
* Если **наименьшее**, то:
    * «x ∈ A» заменяется **0**
    * if ( (условие) **== 0**) { ... }
