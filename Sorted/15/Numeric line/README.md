# Задание 15. На Числовые отрезки
###### Задания с решу егэ и сайта Полякова.
###### C 15-5 идет более совершенный код.

- Если просят найти **максимальную длину**, то:
    - «x ∈ A» заменяется **1**
    - if ( (условие) **== 1**) { ... }
* Если **минимальную**, то:
    * «x ∈ A» заменяется **0**
    * if ( (условие) **== 0**) { ... }
